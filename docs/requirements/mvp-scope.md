# MVP 范围定义

> 状态：已确认
> 更新日期：2026-03-09

---

## MVP 一句话描述

用户输入文字描述和歌词，系统生成 2 首候选歌曲，用户可在线试听、下载、查看历史记录。

---

## 功能范围

### ✅ MVP 必须有

**用户系统**
- 注册 / 登录（**MVP 仅支持邮箱**；手机号短信验证码延后至 Phase 2+）
- 免费额度（每用户每天 N 次生成，N 待定）

**音乐生成（核心价值）**
- 文本描述（prompt）→ 生成歌曲（含人声）
- 纯音乐模式（无人声）
- 自定义歌词输入
- 自动生成歌词（不输入歌词时）
- 每次生成 2 个候选供用户选择
- 生成进度实时反馈（WebSocket）

**内容管理**
- 我的作品库（历史记录列表）
- 在线播放
- 下载 MP3

---

### ⚠️ 第二版再做（MVP 暂缓）

- 歌曲续写 / 延长时长
- Remix（风格改编）
- 社区广场 / Explore（公开作品浏览）
- 分享链接
- 封面图生成
- 专辑 / 播放列表
- 收藏 / 点赞

---

### ❌ 明确不做

- 片段重新生成（Reprompt）
- 参考音频上传（风格迁移）
- 关注 / 粉丝系统
- 订阅付费（先跑通免费流程）
- 商业授权管理

---

## 基础设施选型（已确认）

**首要原则：优先国内服务，支持手机号 + 支付宝/微信注册，无需国际信用卡。**

| 组件 | 选型 | 免费额度 | 备注 |
|--|--|--|--|
| API 服务器 | 阿里云 / 腾讯云 ECS | 新用户免费试用 1-3 个月 | 到期后按量付费或自建 |
| 数据库 | 阿里云 RDS MySQL | 新用户免费试用 | 或自建在 ECS |
| Redis | 自建在 ECS | — | 省一个实例费用 |
| 对象存储 | 七牛云 | 10GB 永久免费 | 含 CDN 免费额度 |
| CDN | 七牛云 CDN | 每月免费额度 | 音频分发 |
| GPU 推理 | AutoDL | 无免费，按小时计费 | 国内最便宜，支持支付宝 |
| 阶段0推理 API | 硅基流动 SiliconFlow | 注册赠送额度 | 替代自建 GPU，验证链路用 |

**推荐注册顺序：**
1. 阿里云（ECS + RDS 免费试用）
2. AutoDL（GPU，最快，手机号注册）
3. 七牛云（对象存储 + CDN，永久免费层）
4. 硅基流动（阶段0推理 API，注册即送额度）

## 成本估算

| 阶段 | 用户规模 | 月成本 | 说明 |
|--|--|--|--|
| 阶段0 | 仅自测 | **¥0** | 硅基流动免费额度 + 各平台免费层 |
| 阶段1 | 10-20人内测 | **¥50-150** | AutoDL 按需开启（RTX 3090 ¥2/h） |
| 阶段2 | 100-200人公测 | **¥400-600** | AutoDL 常驻 + 七牛云超额存储 |

**GPU 是唯一刚性成本，控制策略：**
1. **按需启停**：Redis 队列有任务才开 GPU 服务器，空闲立即关机
2. **模型降级**：MVP 阶段先用 MusicGen-Small，速度快 5-10 倍
3. **限制免费额度**：每用户每天上限 N 次，防止滥用导致费用失控

## 技术风险

| 风险 | 等级 | 应对 |
|--|--|--|
| AudioCraft 商业授权问题 | 🔴 高 | 上线前必须确认 Meta 授权协议，MVP 仅内部使用可规避 |
| 人声 / 歌词对齐质量不稳定 | 🟡 中 | 先接受现有质量，不列为 MVP 阻塞项 |
| GPU 成本不可控 | 🟡 中 | 严格限制每日免费额度，监控用量 |
| 阿里云免费试用到期 | 🟢 低 | 到期前评估用量，决定是否迁移自建 |
