# 非功能性需求

> 状态：已确认
> 更新日期：2026-03-09
> 说明：MVP 阶段目标保守，优先跑通，后续逐步提升

---

## 性能

| 指标 | MVP 目标 | 说明 |
|--|--|--|
| API 响应时间（非生成接口） | P99 < 500ms | 登录、查询作品库等 |
| 音乐生成时间 | P50 < 90s，上限 180s | MusicGen-Small，30s 音频 |
| 音频首帧加载时间 | < 3s | 流式播放，不需全量加载 |
| 下载速度 | 受限于七牛云 CDN | 5MB 文件预期 < 5s |

---

## 并发与容量

| 指标 | MVP 目标 | 说明 |
|--|--|--|
| 同时在线用户 | 50 人 | 内测阶段 |
| 同时生成任务数 | 3 个 | 单 GPU 串行处理 |
| 任务队列最大积压 | 20 个 | 超出拒绝并提示用户稍后重试 |
| 单用户每日生成次数 | 5 次 | 免费额度上限 |
| 作品存储上限 | 无（受限于七牛云 10GB 免费额度） | 超出后评估是否清理 |

---

## 可用性

| 指标 | MVP 目标 | 说明 |
|--|--|--|
| API 服务可用性 | 95% | 内测阶段，允许维护窗口 |
| GPU 服务可用性 | 按需启停，不承诺 SLA | 内测阶段可接受 |
| 计划维护窗口 | 每周一凌晨 2-4 点 | 提前通知内测用户 |

---

## 安全

| 要求 | 实现方式 |
|--|--|
| 传输加密 | 全站 HTTPS，证书自动续期 |
| 身份认证 | JWT，有效期 7 天，支持主动失效 |
| 密码存储 | bcrypt 哈希，不存明文 |
| 暴力破解防护 | 登录失败 5 次锁定 15 分钟 |
| 内容过滤 | 关键词黑名单，覆盖 prompt 和歌词 |
| API 限流 | 每 IP 每分钟最多 30 次请求 |
| 文件访问控制 | 音频文件 URL 带时效签名（有效期 1 小时） |

---

## 输入限制

| 字段 | 限制 |
|--|--|
| prompt | 1-200 字，必填 |
| lyrics | 0-500 字，选填 |
| 生成模式 | 枚举值：`vocal` / `instrumental` |
| 作品名称 | 0-50 字，选填 |
| 邮箱 | 标准邮箱格式，最长 100 字符 |
| 密码 | 8-32 字符 |

---

## 兼容性

| 平台 | 要求 |
|--|--|
| Android | API 26+（Android 8.0+） |
| iOS | iOS 15+ |
| H5 | Chrome 90+、Safari 14+、Edge 90+ |
| 网络 | 支持 4G / WiFi，弱网下降级（停止自动刷新，提示用户） |

---

## 可观测性（MVP 最小化）

| 项目 | 方案 |
|--|--|
| 应用日志 | 结构化 JSON 日志，输出到文件，保留 7 天 |
| 错误监控 | 记录 5xx 错误到日志，人工定期检查 |
| 业务指标 | 每日生成次数、成功率、平均耗时（写入 MySQL，手动查询） |
| 告警 | MVP 阶段不做自动告警，靠人工巡检 |

> 第三阶段引入 Prometheus + Grafana 替代以上方案

---

## 数据保留策略

| 数据类型 | 保留策略 |
|--|--|
| 用户账号 | 永久保留 |
| 生成任务记录 | 永久保留（仅元数据，不含音频） |
| 已保存音频文件 | 永久保留（受存储成本约束，后期评估） |
| 未选择的候选音频 | 任务完成后 24 小时内删除 |
| 失败任务 | 记录保留 7 天后清理 |
| 应用日志 | 保留 7 天 |
